# meme-zone


Сайт для публікації та перегляду мемів з AI-підписами українською мовою.

## 🚀 Демо
> Посилання буде додано після деплою

## 🛠 Технології
- **Backend:** FastAPI + SQLite
- **Frontend:** Vanilla JS + Fetch API
- **AI:** Anthropic Claude API

## 📁 Структура проєкту
## ⚙️ Запуск локально

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
Документація API: http://localhost:8000/docs

### Frontend
Відкрити `frontend/index.html` у браузері або через Live Server у VS Code.

## 👥 Команда
| Роль | Завдання |
|------|----------|
| Backend | FastAPI, ендпоінти, база даних |
| Frontend | HTML/CSS/JS, інтеграція з API |
| Docs/DevOps | Документація, репозиторій, деплой |
